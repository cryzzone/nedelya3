# Система учета заявок на ремонт бытовой техники

## Описание

Веб-приложение для управления заявками на ремонт бытовой техники. Система предназначена для автоматизации процессов приема, обработки и контроля выполнения заявок в сервисном центре.

## Функциональность

### Роли пользователей

- **Заказчик** - создание заявок, просмотр статуса своих заявок
- **Оператор** - прием заявок, назначение мастеров, просмотр статистики
- **Мастер** - работа с назначенными заявками, обновление статусов
- **Менеджер** - полный доступ к системе, управление пользователями, отчеты
- **Менеджер по качеству** - контроль качества, эскалация проблем, QR-коды для отзывов

### Основные возможности

- Создание и управление заявками на ремонт
- Система ролей и прав доступа
- Отслеживание статусов заявок
- Назначение мастеров на заявки
- Комментарии к заявкам
- Статистика и отчеты
- Контроль качества и эскалация
- Генерация QR-кодов для сбора отзывов

## Технические требования

- Node.js 14.0 или выше
- SQLite3
- Веб-браузер с поддержкой JavaScript

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
node src/app.js
```

3. Откройте браузер и перейдите по адресу: http://localhost:3001

## База данных

Система использует SQLite с автоматической инициализацией при первом запуске. Создаются следующие таблицы:

- `users` - пользователи системы
- `requests` - заявки на ремонт
- `comments` - комментарии к заявкам
- `quality_ratings` - оценки качества
- `escalations` - эскалации проблем

## Тестовые данные

При первом запуске система автоматически создает тестовых пользователей:

- Заказчик: login: user1, password: pass1
- Оператор: login: operator1, password: pass1
- Мастер: login: master1, password: pass1
- Менеджер: login: manager1, password: pass1
